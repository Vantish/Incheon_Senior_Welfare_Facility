# 인천광역시 노인복지시설 위치 기반 추천 앱

해당 프로젝트는 인천광역시의 노인복지시설 데이터를 사용해, 사용자의 위치에서 가장 가까운 시설을 찾아주는 Streamlit 앱 소스입니다. 
사용자가 원하면 주변 맛집, 여가시설, 버스정류장 등 추가 정보를 지도에 함께 표시합니다.

# [ 주요 구성 및 세부 기능 ]

# 1. 위치 입력 및 지오코딩
사용자가 도로명 주소를 입력하면 지오코딩, kakao API  을 이용해 위도, 경도로 변환하여 위치 정보를 확보합니다.( 도로명 주소까지가 최선)

# 2. 복지시설 최적 추천 알고리즘
k-Nearest Neighbors (KNN) 알고리즘과 Haversine distance 공식을 이용해, 사용자 입력 위치와 가장 근접한 노인복지시설을 탐색하고 추천합니다.

# 3. 주변 POI 통합 추천
복지시설 반경 내 버스정류장(상위 5개), 인근 맛집, 여가·체육시설 같은 POI(Point of Interest)를  추천해 드립니다.

# 4. 지도 시각화 및 경로 안내
folium 라이브러리를 활용해 주요 추천 시설과 주변 POI들의 위치를 분류별 마커로 표시하고, 경로를 PolyLine 으로 연결하여 시각화 하였습니다

# 5. 인터랙티브 웹 UI
Streamlit 기반으로 주소입력, 시설 분류 선택, 관심 장소 멀티셀렉트 등 다양한 사용자 인터페이스 및 검색 결과를 테이블과 지도로 통합해 제공합니다
